Local Agent Integration

            
Connect this dashboard to your local filesystem to unlock the Autonomous Agent.

          

          
          

            

              
Step 1: Install CLI

              
git clone https://github.com/Niteshagarwal01/cybercentry.git
cd cyber_sentry
python -m venv .venv

# Windows:
.venv\Scripts\Activate.ps1

pip install -e ".[dev,scanners,ui]"

            


            

              
Step 2: Initialize Config

              
cp .env.example .env
# Add the OPENROUTER_API_KEY to your .env

cs init .
